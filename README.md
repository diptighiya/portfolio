# Dipti Ghiya — Portfolio

Personal portfolio site built with Next.js 14, TypeScript, and inline styles.

## Features

- macOS-style floating window UI
- Bottom dock navigation
- About window with bio loaded from JSON
- Resume PDF viewer
- Projects window with live demos
- OS Memory Allocator game — an interactive simulation teaching memory management concepts (First Fit, Best Fit, Worst Fit, GC Pause, OOM Killer, Memory Leaks)

## Stack

- Next.js 14 (App Router)
- TypeScript
- React Icons + Lucide React
- Deployed on Vercel

## Run locally

\```bash
npm install
npm run dev
\```
